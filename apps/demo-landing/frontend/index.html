How the swarm works

Five focused agents hand off to each other so you only hear about deals that actually matter.

  • 🗣️
    Interface Agent Captures natural-language flight requests from chat.
  • 🧭
    Orchestrator Agent Schedules scans and manages provider rate budgets.
  • 🛰️
    Fetching / API Agent Checks live flight offers across multiple providers.
  • 📈
    Analytics Agent Evaluates trends and confirms genuine price drops.
  • 🔔
    Notification Agent Sends useful, well-timed alerts when real deals appear.
